# PE01 - Resume Web App

This is a React application that displays my CV/Resume, including: education, skills, work experience, and projects.

## Input

1. **Personal Information**: Full name (Enzo Nguyen), address, phone number, email
2. **Education**: Degree, institution, duration, GPA
3. **Skills**: Programming languages, web technologies, frameworks, databases, tools
4. **Work Experience**: Job title, company, duration, job description
5. **Projects**: Project name and description

## Process

The processing steps are as follows:

1. The React application starts from `App.js`, bundles the code using Webpack, and opens the browser at `localhost:3000`

2. `Resume.js` organizes the resume information into sections such as Education, Skills, Work Experience, and Projects

3. `Resume.css` applies the layout, fonts, spacing, colors, and page styling

4. The browser displays the complete resume page with formatted sections

## Output

The output is a resume webpage displayed in the browser

Displays name and contact information centrally
Displays education details (degree, institution, and GPA)
Lists skills categorized by group
Displays work experience with descriptions
And projects
